# 🌙 NeuroGrade AI Evaluator

An AI-powered academic answer evaluation system that analyzes student responses (text or handwritten images) and provides detailed scoring across multiple dimensions using LLM + OCR.

---

## 🚀 Features

* 📄 **Text Answer Evaluation**
* 🖼️ **Image-based Answer Evaluation (OCR using Tesseract)**
* 🤖 **LLM-based Scoring (LLaMA via Groq API)**
* 📊 Multi-dimensional scoring:

  * Semantic
  * Relevance
  * Completeness
  * Readability
  * Keywords
  * Consistency
  * Coherence
  * Factual Accuracy
  * Concept Understanding
  * Precision / Recall / F1
* 🎯 Final weighted score calculation
